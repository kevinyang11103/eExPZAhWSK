# 固定资产管理系统简介 java610

## 项目概述

本项目是一款基于SpringBoot的公司固定资产管理系统，采用前后端分离的技术架构，使用Java进行开发。

## 技术栈

- **后端框架**：SpringBoot + Mybatis
- **数据库**：MySQL
- **前端框架**：Vue + Node.js
- **开发工具**：IDEA / Eclipse、Navicat

## 系统角色

- **管理员**：负责平台整体运营和管理。
- **员工**：使用平台进行资产查看、申请、领用及报修等操作。

## 功能模块

- 用户注册/登录
  - 个人信息管理（包括修改密码）
- 系统首页
  - 包含公告和统计图表
- 资产管理
  - 资产分类管理
  - 资产信息管理
  - 资产入库审核
  - 资产领用审核
  - 资产报修审核
- 部门信息管理
- 公告信息发布与管理
- 管理员信息管理
- 员工信息管理
